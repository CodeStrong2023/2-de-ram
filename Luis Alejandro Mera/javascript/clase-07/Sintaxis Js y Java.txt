Secuencia de escape
\' = Mostrar una comilla simple dentro de una cadena de información 
\" = Mostrar una comilla doble dentro de una cadena de información
\\ = Es para mostrar un back slash
\n = Es un salto de linea
\r = Retorno
\t = Tabulación
\b = Retroceso
\f = Salto de página

Operador de aumento
++
// Pre incremento (JavaScript)
Ejemplo: let z = ++a
// Pos incremento (JavaScript)
Ejemplo: let z = a++

Operador de decremeto 
--
// Pre decremeto (JavaScript)
Ejemplo: let z = --a
// Pos decremeto (JavaScript)
Ejemplo: let z = a--

Operador Distinto, Diferente (retornan un tipo booleano)
